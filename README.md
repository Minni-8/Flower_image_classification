# 🌸 Flower Image Classification

## Project Description
This project classifies flower images into five categories:
- Lily
- Lotus
- Orchid
- Sunflower
- Tulip

## Technologies Used
- Python
- TensorFlow
- NumPy
- Matplotlib
- Streamlit

## Dataset
Kaggle Flower Dataset
(https://www.kaggle.com/datasets/kausthubkannan/5-flower-types-classification-dataset)

## Model
Convolutional Neural Network (CNN)

## Validation Accuracy
80.4%

## Features
- Upload a flower image
- Predict the flower species
- Display prediction confidence

## How to Run
1. Install the required libraries.
2. Run:
   streamlit run app.py
